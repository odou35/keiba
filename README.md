# プロジェクト名

## フォルダ構成

- `backend` フォルダ: サーバーサイドの処理。

- `frontend` フォルダ: クライアントサイドの処理。

- `create.js / insert.py`: データの作成。

## 依存関係

以下はプロジェクトが使用する主な依存関係です。

- `jsonwebtoken`: ^9.0.1
- `mysql`: ^2.18.1
- `mysql2`: ^3.5.2
- `nodemon`: ^3.0.1
